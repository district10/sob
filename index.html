1 Emacs 设置

在用 C-x C-e h h 导出 html 前, 配置一下 Emacs: 

(setq org-html-htmlize-output-type 'css)     ;; default: 'inline-css
(setq org-html-htmlize-font-prefix "org-")   ;; default: ""

2 .org 文件的组织

2.1 setup.org

存储博客的设置, 包括 CSS, 一些宏: 

#+AUTHOR: An Org-mode Blogger

#+HTML_HEAD: <link rel="stylesheet" href="css/worg.css" type="text/css" media="screen" />
#+HTML_HEAD: <link rel="stylesheet" href="css/main.css" type="text/css" media="screen" />

#+MACRO: begin-summary #+HTML: <div class="summary"><h2>Summary</h2>
#+MACRO: end-summary #+HTML: </div>

#+MACRO: begin-hidden #+HTML: <div class="hidden"><strong>$1 <small>click to toggle</small></strong><div class="hidden-content">
#+MACRO: end-hidden #+HTML: </div></div>

(worg.css 来自 org-mode 官网.)

2.2 footer.org

放入如下内容: 

-----
#+BEGIN_CENTER
Blog by an Org-mode Blogger.
#+END_CENTER

#+BEGIN_HTML
<script src="http://code.jquery.com/jquery-1.9.1.js"></script>
<script src="http://code.jquery.com/ui/1.10.3/jquery-ui.js"></script>

<script type="text/javascript">
$('.hidden .hidden-content').hide();
$('.hidden > strong').click(function() {
  $(this).parent().find('.hidden-content').toggle();
});
</script>

<script type="text/javascript" src="js/main.js"></script>
#+END_HTML
